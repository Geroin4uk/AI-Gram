Как взять API-ключ Google AI Studio
Подробный безопасный маршрут: от входа в аккаунт до вставки ключа в это приложение.
1. Открой сайт

Перейди на aistudio.google.com в браузере. Лучше использовать обычную вкладку, а не режим инкогнито, чтобы вход в Google сохранился.

2. Войди в Google-аккаунт

Нажми Sign in / Войти и выбери аккаунт. Если Google попросит принять условия Google AI Studio, прочитай их и подтверди.

3. Открой раздел ключей

В верхней или боковой части интерфейса найди кнопку Get API key / Получить API key. Иногда она находится в меню API keys.

4. Создай ключ

Нажми Create API key / Создать API key. Если появится выбор проекта Google Cloud, можно выбрать существующий проект или создать новый.

5. Скопируй значение

После создания скопируй ключ целиком. Он обычно начинается с AIza и показывается один раз или скрывается после закрытия окна.

6. Вернись в настройки ИИ

Нажми кнопку назад вверху этого окна, выбери провайдера Google AI Studio, вставь ключ в поле API Ключ и нажми Сохранить.

7. Проверь модель

Оставь gemini-3.1-flash-lite или впиши другую доступную модель из Google AI Studio. Если модель недоступна, приложение покажет ошибку API и временно ответит локально.

Не публикуй ключ в чатах, GitHub, скриншотах и общих документах. На чужом устройстве не включай сохранение ключа в браузере.
